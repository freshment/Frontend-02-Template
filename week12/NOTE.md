# week12 学习笔记

## 组件
### 组件区别于 对象 和 模块
组件是跟UI强相关的东西，某种意义上可以是 特殊的对象 或者 特殊的模块
组件特点：可以以树形结构组合

```bash
对象
* Properties 属性              
* Methods 方法                 
* Inherit 继承关系              
```

```bash
组件
* Properties 属性
* Methods 方法  
* Inherit
* Attribute 特性
* Config & State
* Event
* Lifecycle
* Children  （树形结构的必要性）
```
